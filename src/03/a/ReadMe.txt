MyPoint p = new MyPoint();
とすべき

3-2
MyPointのインスタンス変数yはprivate変数であり、外部からアクセスすることはできない。
setPointなどのメソッドを利用すべき。

3-3
aという配列を宣言したものの、生成と初期化を行っていない。
int[] a = new int[10];
のように生成を行う必要がある。

3-4
配列のindex26は配列で定義されていない。
int[26]としたときに作られる配列は長さが26であり、
0から始まるindexが26までは存在しない。
indexは25までである。

計算はかっこがない場合左から行われる。
Stringと数値の間に+がある場合は数値が文字列に変換され、結合される。
そのため上二つは文字列として連結される。
一方、かっこをつけた場合は数値＋数値の形式になり、
数値演算として認識されるため、足し算が行われたのちに連結される。
